Про проект

Проект полягав у встановленні та налаштуванні нового сервера DELL, придбаного компанією. Основним завданням було забезпечення оптимальної роботи обладнання для потреб компанії, яке включало кілька ключових етапів:

  1. Інсталяція операційної системи ESXi 7.0 з використанням кастомних драйверів від VMware, спеціально адаптованих для серверів DELL. Процес також включав активацію продукту з використанням ліцензійного ключа.
  2. Фізичне розміщення сервера у спеціалізовану шафу. Підключення і налаштування розумного безперебійника для забезпечення стабільної роботи.
  3. Налаштування сервера, включаючи інсталяцію ОС під наглядом керівника, налаштування віддаленого доступу через веб-сайт, доступ через iDRAC, а також конфігурацію системи RAID 10 для оптимізації зберігання даних.
  4. Створення та налаштування віртуальних машин на основі Windows Server 2022 для різних потреб компанії, включаючи одну машину для створення бекапів через програму Veeam. Керівником було наголошено на важливості ретельного розділення ресурсів між машинами, щоб запобігти майбутнім проблемам у їхньому функціонуванні.
  5. Проведення тестів на відмовостійкість і подальше налаштування системи для забезпечення її надійності і продуктивності.

Результати

Опис та візуалізація результатів роботи, включаючи скріншоти та відео. Нижче представлено кілька знімків, що демонструють встановлення сервера, налаштування віртуальних машин і системи для бекапів.

Відгуки

Знання та навички практиканта в області мереж та системного адміністрування дозволили йому успішно виконувати індивідуальний проект. Серед завдань були такі як: встановлення серверного обладнання DELL EMC R350, інсталяція і подальше налаштування ОС ESXi 7.0, створення чотирьох віртуальних машин із встановленням Windows Server 2022, створення та налагодження ВМ для бекапів, тестування відмовостійкості та працездатності системи.
